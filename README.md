# mw-web
Web
```
Node v18.14.1
```
## Clone and start the web
### Step 1
Clone the git repository
```
git clone https://github.com/mrsasi/mw-web.git
```
### Step 2
Checkout your current project repository like "mw-web"
```
cd mw-web
```
### Step 3
Install the web
```
npm install
```
### Step 4
Start the web
```
npm run dev
```
